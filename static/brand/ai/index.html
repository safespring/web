Usage Rules

These guardrails are intentionally simple so they work across image, slide, document, and diagram tools.

Do

  • Use a restrained European cloud infrastructure style.
  • Use real logo assets from the manifest.
  • Use cloud-blue backgrounds with main-color text and icons.
  • Keep text readable and evidence-led.
  • Use orange for one primary emphasis.

Do Not

  • Invent certifications, customers, claims, or regions.
  • Redraw or distort the Safespring logo.
  • Use main colors as ordinary component or slide backgrounds.
  • Use cyberpunk, cartoon, luxury, or consumer-app styling.
  • Place text over low-contrast imagery.
